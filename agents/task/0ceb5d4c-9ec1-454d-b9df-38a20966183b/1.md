- gap-md-3 → horizontal/vertical spacing between selectors only in row mode (md+); narrow mode keeps child mb-3.
- align-items-md-start → top-align selectors of differing heights in row mode.
- mb-3 on wrapper → spacing below selector block before items list.
- child selector mb-3 retained (component unchanged per constraint) → harmless in row mode (overridden visually by gap+alignment), correct in column mode.
